Reusable UI Foundation

monoorbit-ds

Framework-agnostic foundations, semantic tokens, and a first-party Angular implementation in one workspace.

Foundations

Use the neutral core from any stack via SCSS, CSS variables, or token JSON.

@use 'monoorbit-ds/foundations';
@import 'monoorbit-ds/tokens.css';

Angular Implementation

Angular is the current component adapter, not the entire system.

import { ButtonComponent, InputFieldComponent } from 'monoorbit-ds/angular';

    @for (component of components; track component) {

Styles

The full style layer builds on top of foundations and keeps component classes aligned with the same token contract.

@use 'monoorbit-ds/styles';
