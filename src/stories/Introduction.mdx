# monoorbit-ds

A design system workspace with framework-agnostic foundations and a first-party Angular implementation.

## Features

- **Framework-agnostic foundations** exposed as SCSS, CSS custom properties, and JSON tokens
- **13 standalone Angular components** built on top of those foundations
- **Neutral visual foundation** that can be re-skinned without rewriting component CSS
- **BEM-style** SCSS architecture
- **Accessible** interactions with keyboard navigation and ARIA support
- **ControlValueAccessor** support for form controls

## Getting Started

Import the foundations or the full style layer:

```scss
@use 'monoorbit-ds/foundations';
@use 'monoorbit-ds/styles';
```

Then import Angular components when needed:

```typescript
import { InputFieldComponent, CheckboxComponent } from 'monoorbit-ds/angular';
```
